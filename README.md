# Threads 활동 내역

Threads 포스팅 기록을 Github 잔디처럼 보여주는 프로젝트입니다.

## 실행 방법

1. **의존성 설치:**

   ```bash
   npm install
   ```

2. **환경 변수 설정:**

   `.env.local` 파일을 생성하고 아래와 같이 Threads Access Token을 추가합니다.

   ```
   THREADS_ACCESS_TOKEN=your_access_token
   ```

3. **개발 서버 실행:**

   ```bash
   npm run dev
   ```

4. **브라우저에서 확인:**

   브라우저에서 `http://localhost:3000`으로 접속하여 확인할 수 있습니다.
